## Glossaire

| Terme | Définition |
|---|---|
| **Order Block (OB)** | Dernière bougie de consolidation avant un mouvement institutionnel impulsif. |
| **Fair Value Gap (FVG)** | Espace de prix non comblé entre la mèche haute de la bougie N-1 et la mèche basse de N+1. |
| **BOS** | Break of Structure : cassure de swing dans le sens de la tendance. |
| **CHoCH** | Change of Character : cassure de swing contre-tendance, signal de retournement. |
| **BSL / SSL** | Buy-Side / Sell-Side Liquidity : zones de stops au-dessus/en-dessous des swings. |
| **Liquidity Sweep** | Mouvement bref qui balaie une zone de liquidité avant de retourner. |
| **EQH / EQL** | Equal Highs / Equal Lows : sommets/creux alignés, liquidité concentrée. |
| **Mitigation Block** | OB visité une fois, agit comme S/R au retest. |
| **Breaker Block** | OB dont la structure a cassé, change de rôle. |
| **OTE** | Optimal Trade Entry : retracement Fibonacci 62-79 %. |
| **Killzone** | Plage horaire de forte activité (Asia, London, NY). |
| **Premium / Discount** | Range divisé à 50 % Fibonacci : vendre premium, acheter discount. |
| **Règle du 1 %** | Risquer max 1 % du capital par trade. |
| **R/R** | Risk/Reward : ratio gain potentiel / perte potentielle. |
