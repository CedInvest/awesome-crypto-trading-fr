## Indicateurs Pine Script

Scripts Pine Script (TradingView) utiles pour l'analyse technique et la prise de décision.

| Indicateur | Description | Source |
|------------|-------------|--------|
| **SMC – Structure de marché** | Détecte automatiquement les *Break of Structure* (BOS) et *Change of Character* (CHoCH) selon la méthodologie Smart Money Concepts. | [Rechercher « SMC Structure »](https://www.tradingview.com/scripts/search/?text=SMC+structure) |
| **Niveaux de liquidité** | Identifie les zones de liquidité (equal highs/lows, order blocks) pour anticiper les mouvements institutionnels. | [Rechercher « liquidity levels »](https://www.tradingview.com/scripts/search/?text=liquidity+levels) |
| **RSI Divergence** | Repère automatiquement les divergences haussières et baissières entre le prix et le RSI. | [Rechercher « RSI divergence »](https://www.tradingview.com/scripts/search/?text=RSI+divergence) |
| **EMA Ribbon** | Affiche un ruban d'EMA (8, 13, 21, 34, 55) pour visualiser la tendance et les zones de support/résistance dynamiques. | [Rechercher « EMA ribbon »](https://www.tradingview.com/scripts/search/?text=EMA+ribbon) |
| **Volume Profile** | Représente la distribution du volume par niveau de prix pour identifier les zones à fort intérêt (POC, VAH, VAL). | [Rechercher « volume profile »](https://www.tradingview.com/scripts/search/?text=volume+profile) |

> 💡 Pour trouver et utiliser des indicateurs Pine Script : ouvrez TradingView → *Indicateurs* → *Rechercher dans la communauté*.
